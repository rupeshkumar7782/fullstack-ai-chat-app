# Full Stack AI Chat App

This is a full stack AI chat application where users can chat with an AI in real-time.

## Features

* Real-time chat
* Simple UI
* Fast response

## Tech Stack

* Frontend: React.js
* Backend: Node.js, Express

 Installation

```bash
git clone https://github.com/YOUR_USERNAME/fullstack-ai-chat-app.git
cd fullstack-ai-chat-app
```

Run Frontend

```bash
cd Frontend
npm install
npm run dev
```

 Backend

```bash
cd Backend
npm install
npm start
```
