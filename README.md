# Full Stack AI Chat App

This is a full stack AI chat application where users can chat with an AI in real-time.

## Features

* Real-time chat
* Simple UI
* Fast response

## Tech Stack

* Frontend: React.js
* Backend: Node.js, Express

 Installation

```bash
git --version
cd C:\Users\rupes\OneDrive\Attachments\Desktop\fullstack-ai-chat-app
```

Run Frontend

```bash
cd Frontend
npm install
npm run dev
```

 Backend

```bash
cd Backend
npm install
npm start
```
